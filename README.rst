Introduction
************

API for engineering calculations with a simple syntax. 

Sintax
******

exponent   :: '^'
multiplication  :: '*' | '/'
add  :: '+' | '-'
integer :: ['+' | '-'] '0'..'9'+
atom    :: PI | E | real | fn '(' expr ')' | '(' expr ')'
factor  :: atom [ expop factor ]*
term    :: factor [ multop factor ]*
expr    :: term [ addop term ]*
