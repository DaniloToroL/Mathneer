Quickstart
##########

.. code:: python

    from math_parser import MathParser
    mp = MathParser()
    result = mp.eval('sin(90*pi/180)')
    print(result)
    
    result = mp.eval('exp(2^4)')
    print(result)

    # return 
    # > 1.0
    # > 8886110.520507872
